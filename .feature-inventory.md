# Seedha Plot Feature Inventory

Authoritative list of every user-visible feature, grouped by screen. Every entry has a `grep` pattern: a literal string that MUST exist somewhere in `src/` for the feature to be present.

`scripts/check-features.js` reads this file and verifies. Run `npm run check-features`.

When you add a feature, append here. When you remove one intentionally, delete here. When you land a PR, the pre-commit or CI check will catch drift.

Last audited: 2026-04-17.
