# Changelog

## 3.0.0 (2026-04-29)

Major version bump for a 500-card deck, a v3 voice fingerprint, a restored color palette, and a new shareable Results screen.

### Deck
- Expanded to **500 cards** (up from 355). 253 BW, 247 TW. New cards land across difficulty and era.
- **Voice fingerprint v3** applied. Sharp-grandma-with-a-chaotic-sibling register, two-beat cadence, geography earns its keep, family-safe ceiling.
- 13 KILL voice rewrites resolved. `bw086` Lage Raho Munna Bhai, `bw15` Munna Bhai MBBS, and `bw24` Hera Pheri swapped off the Q4 canon ban (replaced with Bheja Fry, Welcome, Wake Up Sid).
- 48 fact errors fixed against Wikipedia: wrong cities, wrong professions, wrong films generated by an earlier batch.
- 78 C-grade voice polish items deferred to 3.0.1.

### Share
- **Results screen now ships a 6-channel share band**: WhatsApp, X, Reddit, Facebook, Email, Copy.
- Each channel carries `utm_source` per channel and `utm_medium=results_share` for clean PostHog attribution.
- Single tagged URL in the share text, no double-paste.

### UI
- **Restored CLAUDE.md LOCKED palette tokens.** A drift to a "v4 midnight magenta" prototype palette had landed in `style.css`; `--ink` is back to `#1A0F0A` and `--tomato` to `#C8321C`.
- **Mobile gameplay is no longer invisible.** `index.html` now flexes the root div into the column chain so `.v8-game-card-area` renders above zero on iPhone-class viewports.
- `TurnInterstitial` keyboard handler hoisted above the early return (Rules-of-Hooks bug).
- Italic Fraunces no longer shrinks below 14px on long card bodies.
- HomeScreen tagline now reads "We describe desi movies badly" so the Tollywood half of the deck is no longer rhetorically excluded.
- Settings screen actually controls things now: difficulty filter and round length flow into the engine instead of being dead UI.
- Card REPORT moved to the GameScreen mast row.

### Engineering
- 187 / 187 vitest cases pass (was 184; +3 cover Settings + share-dedup contracts).
- typecheck, lint, build all green.
- Bundle 237 KB raw / 70 KB gzipped.
- `puppeteer` added as dev-dep for visual-QA tooling.

### Verification
- Full pre-ship gauntlet ran across 10 phases (baseline, card auditor, fact checker, voice and fact rework, visual QA, visual rework, 50-persona player simulator, 7-reviewer panel, holistic rework, consolidation). Reports archived under `.audits/2026-04-28/`.
